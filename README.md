# 💇‍♀️ Glamour & Style Salon Website Template

A modern, responsive, and mobile-friendly salon website template built for beauty parlors, hair salons, and barbershops.

---
 Features
**Naira Pricing:** Pre-configured with realistic Nigerian salon service prices.
**Style Gallery:** Image grid to showcase hairstyles, cuts, and wig revamps.
**Customer Reviews:** Testimonial section with star ratings to build trust.
**Appointment Form:** Simple online booking request form.
**Fully Responsive:** Looks great on mobile phones, tablets, and desktop screens.

---

 Built With
**HTML5** - Structure and layout
**CSS3** - Styling, custom colors, and responsive grid flexbox
**JavaScript** - Form submission handling
